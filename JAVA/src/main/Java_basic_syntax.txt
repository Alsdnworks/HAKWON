자바의 기초 문법 구문

메소드로 구성된 클래스, 
컴파일된 클래스나 서로 관련된 클래스를 모아 패키지가된다.
패키지는 모여 프로젝트를 형성한다.

따라서 클래스의 첫부분에서 패키지를 명시하고
라이브러리를 호출(import)
이하 클래스 선언후
메소드를 작성하게 된다

C로 설명하자면
cpp파일들이있는 폴더는 패키지로 명시되고 
stdio.h 라이브러리를 호출 한 후
파일명(클래스)선언후 int main에 들어가는 구조로 볼 수있다.


package 패키지명;
(임포트)
public class 파일명 {
	public static void main(string[] args){
	}
}
